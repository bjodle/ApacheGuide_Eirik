Part 1

First you're going to create a new Virtual Machine.
Except for filling out what you want your Virtual Machine tobe called you can skip until you see this screen, where you will switch from "Windows" to "Linux".
Then scroll to the bottom and select "Ubuntu Linux (64-bit)", then press next.
Under "New Network" press "dummy_temp_vswitch" and click bowse.
Now select the user provided by your teacher.
Under the catagory "New Hard disk" change disk provision from thick to thin.
On "New CD/DVD Drive" change "Client Devise" to "Content Library ISO File".
Select the first file titled ubuntu-20.04.3-live-server-amd64.
Make sure that thers a check on the box that says "connect..."
Now change to the "VM Options".
Under Boot Options check the box under "Force BIOS setup".
Double check that you have followed all the steps correctly and continue until you see this screen and power on you VM.
Press Launch.

Part 2

Using the arrow keys cycle over to "Boot" and select "Network boot from VMware VMXNET3" with your enter key.
Cycle over to Exit and save your changes.
You can now exit by pressing F10 or by navagating over to "Exit Saving Changes".
Press next until you reach this screen.
Under "ens160" select "Edit IPv4".
Set the IPv4 Method to Manual.
Fill out the highlighted boxes with the information by your teacher.
When you are ready continue.
Fill out the boxes with the information you want.
When you get to this screen make sure that this box is checked.
After a while of installing press "Reboot Now".

Part 3

Enter using the login info you made.
Write "sudo su" and enter your password.
This isn't necessary however this will make your machine recognise you so you don't have to type your password anymore.
Write "sudo apt update"
Write "sudo apt install apache2"
Write "apache2 -version".
You should get something resembling this.
Write "sudo ufw app list"
You should see the available applications.
Write "sudo ufw allow 'Apache'".
It should tell you that the rules are updated.
Write "sudo ufw status" status should show: inactive
Write "sudo systemctl status apache2".
Under "Loaded" it should say active (running) in green letters.
Write "hostname -I"
Write "sudo mkdir -p /var/www/(your domain).net/html".
Write "sudo chown -R $USER:$USER /var/www/(your domain).net/html".
Write "sudo chmod -R 755 /var/www/(your domain).net".
Write "nano /var/www/(your domain).net/html/index.hmtl".

Part 4

Write what is displayed in the photo, you can replace the text with what you want, save by pressing ctrl+O and exit using ctrl+X.

Write what is displayed in the photo but replace "info" with your domain.
Write "sudo a2ensite (your domain).net.conf"
